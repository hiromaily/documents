# GoReplay

GoReplay は、実際のデータを使用してシステムを継続的にテストするために、ライブ HTTP トラフィックをキャプチャしてテスト環境に再生するオープンソース ツール。コードの Deploy、構成の変更、インフラストラクチャの変更に対する信頼性を高めるために使用できる。
