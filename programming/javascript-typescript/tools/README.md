# Tools

Typescriptで書かれたファイルは`compile`, `bundle`, `minify` されなくてはならない

- General Build Tools
  - Vite
  - [Rome](https://rome.tools/)
- Bundlers
  - esbuild
  - Webpack
  - Rollup
  - Browserify
- Compilers/Transpilers
  - SWC (Bundlerとしても使える)
  - Babel (JavaScript transpile)
  - SASS/LESS/Stylus (CSS preprocessors)
- Module Loaders
  - RequireJS
