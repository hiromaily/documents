# HTTP Status Code For Redirect

## 301 Moved Permanently

- Do cache
- It's possible to change from POST to GET

## 302 Found

- Not cache
- It's possible to change from POST to GET

## 307 Temporary Redirect

- Not cache
- It's not possible to change from POST to GET

## 308 Moved Permanently

- Do cache
- It's not possible to change from POST to GET
