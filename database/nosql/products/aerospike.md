# Aerospike

特に低レイテンシ(通信遅延)と高スループットを必要とする環境において、リアルタイムのトランザクション処理を処理するように設計された、高性能でスケーラブルな分散型 NoSQL データベース。
強力な一貫性、ハイブリッド・メモリ・アーキテクチャ、マルチサイト・クラスタリングなどの高度な機能により、広告技術、金融サービス、ゲームなどの業界におけるリアルタイム・アプリケーションに最適な選択肢となる。

## 特徴

### 高いパフォーマンス

スピードに最適化されており、ミリ秒以下のレイテンシで毎秒数百万件のトランザクションを処理できる。 広告技術、金融サービス、オンラインゲームなど、パフォーマンスが極めて重要なユースケース向けに設計されている

### ハイブリッド・メモリ・アーキテクチャ

データストレージに `RAM` と `SSD(または NVMe)`の両方を使用する独自のハイブリッド・メモリ・アーキテクチャを採用している。 頻繁にアクセスされるデータは `RAM` に保持され、アクセス頻度の低いデータは`ディスク`に保存されるため、高速性と大容量の両方を実現できる

### データの一貫性

ノード間およびデータセンター間の`同期レプリケーション`のサポートを含む強力な一貫性モデルを提供しており、データの信頼性の高い保存とアクセスを保証する

### スケーラビリティ

`水平方向`に拡張できるため、クラスタにノードを追加して、増加したデータ量やトラフィックに対応することができる。このスケーラビリティは自動的かつシームレスであり、運用上のオーバーヘッドは最小限に抑えられる

### 高可用性

`レプリケーション`と`自動フェイルオーバー`をサポートしており、ノードに障害が発生した場合でもデータの可用性を確保する。このデータベースは、ミッションクリティカルなアプリケーションに不可欠な高可用性と稼働率を維持するように設計されている

### 豊富なデータモデル

そのコア部分では KeyValue Store だが、`リスト`、`マップ`、`ユーザー定義関数(UDF)`などのより複雑なデータタイプもサポートしており、より高度なデータ処理が可能

### マルチサイトクラスタリング

データセンター間の複製(XDR)をサポートしており、地理的に離れた地域間でデータを複製するマルチサイトクラスタの作成が可能。これは、世界各地からの低レイテンシアクセスを必要とするグローバルアプリケーションに有用

### 耐久性

Aerospike 内のデータはディスクに永続化でき、データベースは`write-ahead logging`をサポートしているため、クラッシュが発生した場合でもトランザクションの耐久性と回復性が確保される。

### 運用上の簡素性

自動データ再バランシング、最小限の構成要件、包括的な監視ツールなどの機能により、管理が容易になるように設計されている

## 一般的なユースケース

### 広告テクノロジーにおけるリアルタイム入札(RTB)

広告テクノロジーにおける、広告掲載の高速オークション管理

### 不正行為の検出

金融サービスにおける、不正行為の検出と防止にリアルタイム分析

### セッション管理

オンラインゲームや e コマースのプラットフォームにおいける、大規模なユーザーセッションの管理

### レコメンデーションエンジン

大量のユーザーデータを迅速に取得・処理することで、レコメンデーションシステムを強化する

## Aerospike と従来のキーバリュー型ストアとの比較

Aerospike は、シンプルな Key Value 型ストアとは異なり、高度な機能(強力な一貫性、複雑なデータ型、ハイブリッド・メモリ・モデルなど)を提供する。これにより、基本的なキーバリュー検索以上の機能が必要なアプリケーションに適しており、特にパフォーマンス、スケーラビリティ、可用性が重要な場合に適している
