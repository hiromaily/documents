## [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

- local dev server を立ち上げる

### 手順

- ディレクトリを作成し、中に json ファイルをセット
- `Command Palette` => `Shift + Command + P` or `F1`
  - `Live Server`と入力
