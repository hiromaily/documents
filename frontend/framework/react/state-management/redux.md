# Redux
Reduxは、Flixアーキテクチャーに基づき`Action`と呼ばれるイベントを使用して、アプリケーションの`State`（状態）を管理し更新するためのフレームワーク。 アプリケーションの多くの部分で必要な`State`を、グローバルなStateとして一元管理するのに役立つ。

これは、React、jQuery、Angular、Vueといったライブラリと併用できる
