# SWR
- データ取得のための React Hooks ライブラリ
- SWR は、まずキャッシュからデータを返し（stale）、次にフェッチリクエストを送り（revalidate）、最後に最新のデータを持ってくる
- Next.jsと同様、Vercelによって開発されている
