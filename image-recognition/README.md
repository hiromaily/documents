# 画像認識 / Image Recognition

## 画像認識技術の種類

1. **伝統的な画像処理技術**:
   - **エッジ検出**（例: Sobelフィルター、Cannyエッジ検出）
   - **テンプレートマッチング**: 特定のパターンや形状を識別するための技術
   - **特徴量抽出**: SIFT、SURF、ORBなど

2. **機械学習（ML）によるアプローチ**:
   - **サポートベクターマシン (SVM)**: 画像分類に使用
   - **k近傍法 (k-NN)**: 画像内のオブジェクトやパターンを識別

3. **深層学習（ディープラーニング）**:
   - **畳み込みニューラルネットワーク（CNN）**: 画像識別のゴールドスタンダード
     - **LeNet**: 初期のCNNモデル
     - **AlexNet**: ILSVRC 2012で広く注目を浴びたモデル
     - **VGGNet**: シンプルで深いモデル
     - **ResNet**: 残差学習により非常に深いネットワークを可能に
     - **Inception (GoogLeNet)**: モジュール構造を持つ効率的なモデル
   - **セグメンテーション**: 画像内の各ピクセルを分類
     - **U-Net**: 医療画像解析で広く使用
     - **Mask R-CNN**: オブジェクト検出とインスタンスセグメンテーション

4. **トランスフォーマーベースのモデル**:
   - **Vision Transformers (ViT)**: 画像認識にトランスフォーマーアーキテクチャを適用
   - **DETR (DEtection TRansformers)**: トランスフォーマーを使用したオブジェクト検出モデル

5. **GenAI & Diffusion Models**:
   - **DALL-E**: テキスト指示による画像生成
   - **Stable Diffusion**: 高品質な画像生成を目的としたディフュージョンモデル

6. **領域特定技術**:
   - **顔認識**: 顔の検出と識別 (例: MTCNN, FaceNet)
   - **OCR (Optical Character Recognition)**: 画像中のテキストを抽出し、デジタルテキストに変換

7. **異常検知とセキュリティアプリケーション**:
   - **異常検知**: 異常なパターンや不正検出
   - **街頭監視システム**: 不審者や物体の検出
