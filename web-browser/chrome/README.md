# Chrome

## Tab

- タブを固定する機能
  - タブの右クリックメニューから`固定`を選ぶと、タブが一番左側に寄せられ、そこに固定された状態になる
- タブをグループ化してコンパクトに
  - グループ化したいタブを選択し、右クリックメニューの`タブをグループに追加` > `新しいグループ` もしくは既存のグループをクリック
